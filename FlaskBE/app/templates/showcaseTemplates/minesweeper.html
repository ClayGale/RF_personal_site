Minesweeper Clone

The project is a full recreation of Minesweeper with some extra features that I tried my hand at making. I added configurable difficulty settings and ridiculously large possible board sizes board sizes.

This project started out as an assignment for COSC 121 - Computer Programming 2. But I far surpassed the requirements of the assignment and am proud of this project considering my knowledge at the time that I made it.

I would likely design aspects of it differently today now that I have a stronger understanding of data-structures and optimization.

DISCLAIMER: This program depends on the JavaFX library. Which was removed from the JDK in version 11. Versions 8 through 10 should run it just fine. At a future date I intend to come back and fix it.
